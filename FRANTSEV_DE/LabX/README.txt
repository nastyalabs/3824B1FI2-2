Консольное приложение, симулирующее работу приложения <Контакты>

Особенности:

По какой-то причине не воспринимает русский язык на ввод

Список контактов отображается над главным меню

Доступны 4 опции:
1) Добавить контакт
2) Редактирование
3) Стереть контакт
4) Найти контакт
0) Выйти из приложения

Сортировка контактов по имени проводится в реальном времени после каждого действия

При добавлении контакта обязательны такие поля, как ФИО и Номер телефона. Почта, организация и должность могут оставаться пустыми

Для удаления контакта требуется указать его номер, который можно увидеть либо в общем списке, либо при поиске

Для поиска контакта требуется указать порядок символов, содержащийся в любом поле нужного контакта

При редактировании контакта если ввести * - поле останется нетронутым

Также чтобы выйти из меню редактирования контакта на выборе номера можно ввести 0
